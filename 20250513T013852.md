Below is the **right‑hand stack**--six _non‑overlapping_ tactical frameworks, one per finger, expressed in the same hard‑edged style as your left‑hand models. Each framework is acronym‑driven for instant recall and ends with a concrete artifact.

* * *

## 1. **Self‑Management -- A L I G N**

> _Objective: Turn personal chaos into a continuously calibrated control loop._

| Step | Action | Instrument | Leading Metric | 
| ---- | ---- | ---- | ----  |
| **A -- Audit** | Weekly inventory of time, energy, and commitments. | 90‑min Sunday Audit | % hours mapped | 
| **L -- Limit** | Impose hard ceilings on WIP tasks and inputs. | WIP cap board | Tasks ≤ 7 | 
| **I -- Intent** | Convert annual goals into 13‑week OKR sprint. | OKR sheet | 80 % KRs on track | 
| **G -- Gear** | Match tasks to chronotype & energy zones. | Calendar colour‑coding | Peak‑energy alignment ≥ 70 % | 
| **N -- Nudge** | Daily micro‑adjust (kill, delegate, automate). | 5‑min evening reset | < 5 min inbox | 

**Core Artifact:** _ALIGN Console_--one Notion page showing WIP cap board, OKRs, and calendar heat‑map.
* * *

## 2. **Problem‑Solving -- O S D C (canonical)**

> _Objective: Convert ambiguous prompts into defensible answers at speed._

| O -- Orient | S -- Structure | D -- Drive | C -- Communicate | 
| ---- | ---- | ---- | ----  |
| Clarify true problem | Build MECE issue tree & hypotheses | Test, model, prove/kill | Synthesize into slides & narrative | 

**Core Artifact:** _OSDC Deck Skeleton_--problem statement, tree, proofs, recommendation.

_(You already wield OSDC; no new acronym invented here--there is no better short‑cycle engine for pure problem‑solving.)_

* * *

## 3. **Action & Execution -- S T R I K E**

> _Objective: Translate strategy into shipped outcomes under deadline pressure._

| Step | Action | Discipline | KPI | 
| ---- | ---- | ---- | ----  |
| **S -- Slice** | Decompose deliverable into irreducible tasks. | Task atomizer | Task granularity ≤ 4 h | 
| **T -- Time‑Block** | Allocate deep‑work blocks; protect calendar. | Zero‑meet AM blocks | ≥ 2 × 90 min/day | 
| **R -- Run** | Execute with single‑piece flow (no multitasking). | Pomodoro × 4 | Flow hours/day | 
| **I -- Identify Blockers** | Surface friction instantly; escalate or solve. | Kanban "Block" lane | Block‑age < 24 h | 
| **K -- Kill Scope‑Creep** | Ruthlessly defer non‑critical extras. | "Kill‑switch" list | Scope delta < 10 % | 
| **E -- Evaluate** | End‑of‑day throughput review → adjust plan. | STRIKE log | Planned vs. done ≥ 85 % | 

**Core Artifact:** _STRIKE Board_--Kanban with Block lane and daily throughput stats.
* * *

## 4. **Leadership -- G U I D E**

> _Objective: Orchestrate people toward a vision with minimal entropy._

| Step | Action | Mechanism | Lag Metric | 
| ---- | ---- | ---- | ----  |
| **G -- Ground** | Articulate non‑negotiable purpose & principles. | 1‑page manifesto | Team recall ≥ 90 % | 
| **U -- Unify** | Map incentives, align via shared KPIs. | Joint scorecard | KPI overlap ≥ 80 % | 
| **I -- Illuminate** | Broadcast progress transparently (dashboard, huddles). | Weekly metrics review | Surprises/month ≤ 1 | 
| **D -- Decentralize** | Push decisions to edge with guardrails. | Delegation matrix | < 48 h decision lag | 
| **E -- Elevate** | Coach & rotate talent into stretch roles. | 1:1 growth plans | Succession cover ≥ 1 deep | 

**Core Artifact:** _GUIDE Charter_--manifesto, scorecard, delegation matrix, succession grid.
* * *

## 5. **Communication -- V O I C E**

> _Objective: Transfer ideas so crisply they survive enemy interrogation._

| Step | Action | Tool | Quality Gate | 
| ---- | ---- | ---- | ----  |
| **V -- Validate Audience** | Identify what they know, fear, want. | Persona sheet | Relevance test pass | 
| **O -- Outline** | Build Pyramid: conclusion → supporting logic. | Slide storyline | MECE check | 
| **I -- Illustrate** | Choose highest‑bandwidth visual (chart, story, demo). | Single‑message slides | 1 insight/slide | 
| **C -- Call‑to‑Action** | State exactly what you want them to do. | CTA banner | Action clarity score ≥ 9/10 | 
| **E -- Echo** | Loop feedback; iterate message until retention. | 24‑h follow‑up ping | Recall ≥ 80 % | 

**Core Artifact:** _VOICE Packet_--persona sheet, storyline, slides, CTA memo, feedback log.
* * *

## 6. **Reflection -- D E L T A**

> _Objective: Capture experience → distill lessons → reprogram behaviour._

| Step | Action | Prompt | Evidence of Success | 
| ---- | ---- | ---- | ----  |
| **D -- Document** | Log events within 24 h (fact, feeling, fallout). | "What just happened?" | Zero memory lag | 
| **E -- Extract** | Identify root causes & patterns. | 5 Whys + pattern tag | Pattern bank growth | 
| **L -- Learn** | Articulate principle or heuristic. | "Therefore, I will…" | Principle library | 
| **T -- Test** | Apply principle in next relevant scenario. | Small‑scale A/B | Behaviour delta | 
| **A -- Archive** | Store lesson in searchable knowledge base. | Zettelkasten node | Retrieval < 30 sec | 

**Core Artifact:** _DELTA Journal_--event log, principle library, tagged retrieval system.
* * *

### **Integrated Flow (Right‑Hand Stack)**
    
    
    ALIGN (self)  →
           STRIKE (execution) →
                  OSDC (problem‑solve) →
                         GUIDE (lead) →
                              VOICE (communicate) →
                                   DELTA (reflect) ─┐
               └──────────────────────────────────────┘  feedback into next ALIGN cycle
    

- **ALIGN** stabilises the operator.
- **STRIKE** converts intent into shipped work.
- **OSDC** unlocks thorny obstacles mid‑stream.
- **GUIDE** scales success through others.
- **VOICE** ensures the narrative compels action.
- **DELTA** codifies insight, feeding the next alignment sprint.

Memorise the acronyms, execute the moves, and your right hand becomes a _precision instrument_ for self‑command, delivery, and influence--mirroring the left‑hand resilience stack you already own.
